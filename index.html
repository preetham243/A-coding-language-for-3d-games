Welcome to uniscript's official website!

Uniscript is a DSL scripting language designed for making games and programs easy and fun. Uniscript was coded in python. it was named after the universe, because you can code a game or "universe" in it.

Why choose Uniscript?

Uniscript was designed to make coding easy, simple, and fun. to make kids to learn text code as easy, and not "hard." to make coding games to so hard. our mission is to promote schools across the US to use text code(like us) and to make game making simple

Features of Uniscript:

-Easy to learn syntax

-Built-in 3D game engine(ursina)

simple to fix bugs in code

Get started with Uniscript today!

To get started with uniscript, download the ursina game engine and python from their official websites. then, you can start coding in uniscript by following this basic template:

input=draw(tree)+[assign_player+view=first person]+{controls=w,s,a,d}

input tells the interpreter what to do,=draw tells to draw, (tree) is the object to draw, [assign_player+view=first person] assigns the player and sets the view to first person, {controls=w,s,a,d} sets the controls for movement.

Command 2:

input=draw.spawn(mob.type=bird)

input assigns the code, = tells what to add, draw.spawn tells it to spawn a mob, (mob.type=...) declares the mob type
